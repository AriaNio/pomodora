# 软软番茄钟

一个纯本地运行的番茄钟小应用，不依赖网络，也不需要安装额外包。

## 使用方式

直接双击打开 `/Users/arianiu/Documents/Playground/index.html` 即可使用。

如果你更习惯本地服务器，也可以在当前目录运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 功能

- 预设 `30 mins`、`15 mins`、`5 mins`
- 自定义 `时 : 分` 选择
- 同一时间只允许一个倒计时
- 切换到其他倒计时时自动重置当前倒计时
- 支持开始、暂停、停止、重置
- 支持声音开关
- 倒计时结束时播放本地提示音，并触发 Q 版角色动画
